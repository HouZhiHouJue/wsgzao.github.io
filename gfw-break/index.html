前言

近日受国家安全政策影响，Google相关服务基本处于全面被墙状态，虽然限制互联网自由本身是不合理，但以发展的眼光去认真思考下GFW的作用或许可以成为一把双刃剑，本文主要结合实际分享简单好用的方法，如果有更优的解决方案我也会持续更新，也希望大家积极反馈，ok, let it go

免费方案

注意 Windows/Linux/MAC/IOS/Android/WP的解决方案都不唯一，请认真阅读原文中的说明部分

Goagent

a gae proxy - Google Project Hosting

https://code.google.com/p/goagent/

Goagent_Assistant

Goagent自动下载、安装、更新、配置工具
【推荐理由】简单可依赖

https://bintray.com/zane/Goagent/Goagent_Assistant/

Buckyball

轻松访问Twitter，Facebook，Youtube的浏览器
【推荐理由】感谢作者的付出

https://code.google.com/p/buckyball/

shadowsocks

A fast tunnel proxy that help you get through firewalls
shadowsocks 是一个可穿透防火墙的轻量代理。
【推荐理由】建议使用shadowsocks-gui

https://github.com/clowwindy/shadowsocks

https://github.com/shadowsocks/shadowsocks-gui

ShadowSocks公益组织

fqrouter

手机变身翻墙路由器，从此科学上网无压力。
【推荐理由】Android首选翻墙方案

http://fqrouter.com/

MAC/IOS/WP

直接使用shadowsocks吧，WP暂时没有简单的方案除了APNP/VPN以外

http://shadowsocks.org/en/download/clients.html

付费方案

注意 请根据自身实际需求，在认真阅读网站说明后做出选择

vpnso

稳定靠谱，良好的口碑

https://killwall.com/

曲径

  • 曲径是一个音乐加速服务。它技术先进，但又简单易用
  • 简单说，它能让你在中国收听Pandora、Spotify和last.fm
  • 有很多热爱生活、具备良好品味的人都在使用曲径

https://getqujing.com/

Green网络加速器

【推荐理由】我不是很喜欢ta的商业推广模式，但性价比高

https://www.grjsq.me/
